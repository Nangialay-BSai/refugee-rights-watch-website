# Refugee Rights Watch Global Website

A Node.js website for Refugee Rights Watch Global organization.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Visit: http://localhost:3000

## Development

```bash
npm run dev
```

## Features

- Responsive design
- Clean, accessible layout
- Express.js server
- Static file serving
